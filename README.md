 📦 Inventory System

A Python-based inventory management system designed for small businesses to efficiently manage products and stock.



 🚀 Features

- ➕ Add new products  
- ✏️ Update existing products  
- ❌ Delete products  
- 📊 Track inventory levels  


 🧠 Project Overview

This project was developed as part of a **Software Fundamentals course**, applying:

- Agile methodology (Scrum)  
- User Stories  
- Incremental development  

The system focuses on solving real business needs by improving inventory organization and control.


## 🛠️ Technologies

- Python  


 🎯 Purpose

The goal of this project is to provide a simple and functional inventory solution for small businesses, combining technical development with a business-oriented mindset.
